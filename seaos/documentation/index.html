Networking

Networking is an ongoing project, but current implemented is a socket layer, an interface layer, and some protocols. Configuring interfaces is done with the ifc command. It has fairly simple syntax:

ifc -i nd1 -s up -n 10.0.0.1/8
ifc -i nd2 -s down
any of the options provided are optional, save for the -i interface-name part. Running ifc without arguments will print out status information of the interfaces on the system.

Additionally, the routing table can be modified by using the route. Running route without arguments will print out the routing table. Routes can be modified like:

route -a -i nd1 -n 10.0.0.0/8 Adds a route to 10.0.0.0/8 via interface nd1.
route -a -i nd1 -n 0.0.0.0 -g 10.0.0.1 -d Adds a default route through 10.0.0.1.
route -r -i nd1 -n 10.0.0.0 Deletes route 10.0.0.0 via nd1.
